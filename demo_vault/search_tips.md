# Search Tips

The knowledge base supports both keyword and semantic search.

## Keyword Search
Match exact terms in note content and titles.
Use quotes for phrase matching: `"exact phrase"`

## Semantic Search
Finds related content by meaning, even when exact words don't match.
Works via embeddings and vector similarity.

## Wikilinks
Type `[[` to see auto-complete suggestions when creating links.
